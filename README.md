# Campus-Placement-Data-Cleaning-Data-Analysis
Campus Placement Data Cleaning &amp; Analysis | Python, Pandas Cleaned, transformed, and analyzed a campus placement dataset using Pandas by handling missing values, duplicates, and outliers, followed by exploratory data analysis to identify placement and salary trends.

Project Overview

This project focuses on cleaning and analyzing a campus placement dataset using Python and Pandas. The primary objective was to improve data quality through preprocessing techniques and perform exploratory data analysis to uncover patterns and trends in student placements.

Tasks Performed
Data Cleaning
Missing Value Detection
Missing Value Imputation using Median
Duplicate Record Checking and Removal
Outlier Detection using IQR Method
Outlier Treatment using Clipping
Data Analysis
Placement Status Analysis
Gender-wise Placement Analysis
Salary Analysis
Specialization-wise Analysis
Work Experience Analysis
Degree Stream Analysis
